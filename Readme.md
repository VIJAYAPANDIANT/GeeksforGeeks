## 🛠 Detailed Solutions

Categorized logic and complexity for each problem.

### 🔹 Array

<details>
<summary>Day 1: Second Largest</summary>

- **Problem:** Find the second largest element in a positive integer array.
- **Logic:**
  1. Initialize `largest = -1`, `secondLargest = -1`.
  2. One pass: Update `largest` and `secondLargest` accordingly.
- **Complexity:** $O(n)$ Time, $O(1)$ Space.
</details>

<details>
<summary>Day 2: Reverse an Array</summary>

- **Problem:** In-place array reversal.
- **Logic:** Two-pointer swap.
- **Complexity:** $O(n)$ Time, $O(1)$ Space.
</details>

<details>
<summary>Day 3: Move All Zeroes to End</summary>

- **Problem:** Shift 0s while preserving order.
- **Logic:** Fill non-zeroes first, then pad remaining indices with 0.
- **Complexity:** $O(n)$ Time, $O(1)$ Space.
</details>

<details>
<summary>Day 4: Rotate Array</summary>

- **Problem:** Counter-clockwise rotation by `d` steps.
- **Logic:** Reversal algorithm (reverse [0..d-1], [d..n-1], then all).
- **Complexity:** $O(n)$ Time, $O(1)$ Space.
</details>

<details>
<summary>Day 5: Next Permutation</summary>

- **Problem:** Lexicographically next greater permutation.
- **Logic:** Find pivot, swap with smallest larger value to right, reverse suffix.
- **Complexity:** $O(n)$ Time, $O(1)$ Space.
</details>

<details>
<summary>Day 6-10: Max Profits & Subarrays</summary>

- **Kadane's Algorithm:** Tracking local and global max sums.
- **Stock Buy/Sell:** Optimized greedy or peak-valley approach.
- **Majority Element:** Boyer-Moore Voting algorithm.
- **Max Product Subarray:** Tracking max/min prefixes.
</details>

<details>
<summary>Day 11-17: Optimized Array Problems</summary>

- **Missing Number:** Index-marking/Sum/XOR methods.
- **Sliding Window:** Maximize 1s, Subarray Sums.
- **Circular Sum:** Total sum - Min subarray sum.
</details>

### 🔹 String

<details>
<summary>Day 18: Anagram</summary>

- **Logic:** Frequency array (size 26) counting.
- **Complexity:** $O(n)$ Time, $O(1)$ Space.
</details>

<details>
<summary>Day 19-21: Pattern Matching & Palindromes</summary>

- **Non-Repeating Char:** Frequency map + double traversal.
- **KMP Algorithm:** Using LPS array for $O(n)$ pattern matching.
- **Binary Addition:** Right-to-left carry tracking.
</details>

<details>
<summary>Day 22-28: String Manipulations</summary>

- **Atoi Implementation:** Handling edge cases (overflow, signs).
- **CamelCase:** Regex or manual upper-case extraction.
- **String Repeats:** Mathematical validation of substring presence.
</details>

### 🔹 Dynamic Programming & Graphs

<details>
<summary>Day 42: Number of Valid Parentheses</summary>

- **Equivalent to:** Catalan Number $C_n = \frac{1}{n+1} \binom{2n}{n}$.
- **Method:** Tabulation for optimal sub-structure.
- **Complexity:** $O(n^2)$ Time, $O(n)$ Space.
</details>

<details>
<summary>Day 41: BFS of Graph</summary>

- **Method:** Queue-based level traversal.
- **Complexity:** $O(V+E)$.
</details>
